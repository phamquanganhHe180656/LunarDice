# Assets — Images

Place the six animal face images here with the following filenames:

| Animal   | Vietnamese | Filename       |
|----------|------------|----------------|
| Gourd    | Bầu        | `gourd.png`    |
| Crab     | Cua        | `crab.png`     |
| Shrimp   | Tôm        | `shrimp.png`   |
| Fish     | Cá         | `fish.png`     |
| Chicken  | Gà         | `chicken.png`  |
| Deer     | Nai        | `deer.png`     |

Recommended size: **256×256 px**, transparent background.

Until real art assets are provided, the app displays emoji as fallbacks.
